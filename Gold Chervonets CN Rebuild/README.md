# Gold Chervonets CN Rebuild

这是一个基于公开资料整理并重写的 MT4 黄金 EA 项目。项目目标是把 `Gold Chervonets` 公开介绍中能确认的策略结构、参数分组和交易逻辑，重写成一份可阅读、可编译、可继续调参的 MQL4 源码。

> 重要说明：本项目不是对原版 `.ex4` 的反编译，不包含原作者源码，不保证与原版逐笔一致。它是面向研究、回测和二次开发的清洁重写版本。

## 策略定位

公开资料显示，原策略主要用于：

- 平台：MetaTrader 4 / MetaTrader 5
- 品种：`XAUUSD`
- 周期：`M15`
- 交易方式：两套系统组合运行
- 核心过滤：ATR 波动过滤、点差过滤、交易时间过滤、账户回撤过滤
- 仓位方式：固定手数、按风险百分比计算手数、可选恢复模式

本重写版优先实现 MT4 版本，并围绕 `XAUUSD M15` 做默认参数和点值换算。

## 系统一逻辑

系统一是双向交易系统，可以做多，也可以做空。公开资料描述它主要使用：

- `Moving Average`，默认周期参考公开介绍中的 55
- `Williams Percent Range`
- `Commodity Channel Index`
- `ATR` 作为波动过滤

### 买入条件

系统一买入大致围绕三个条件：

1. 上一根 M15 K 线收盘价位于均线上方。
2. 当前价格相对上一根收盘价出现回撤。
3. `WPR` 或 `CCI` 进入超卖区域。

当这些条件同时满足，并且 ATR、点差、时间、账户回撤等总过滤允许开仓时，系统一尝试开多。

### 卖出条件

系统一卖出逻辑与买入相反：

1. 上一根 M15 K 线收盘价位于均线下方。
2. 当前价格相对上一根收盘价出现反抽。
3. `WPR` 或 `CCI` 进入超买区域。

如果参数 `系统一只做多` 设置为 `true`，系统一会禁用卖出信号。

### 系统一出场

系统一包含三类出场方式：

- 固定止损和固定止盈
- 达到盈利距离后的追踪止损
- 基于 WPR、M15 收盘价、M1 蜡烛方向和盈利距离的信号平仓

## 系统二逻辑

系统二是只做多系统。公开资料描述它主要使用：

- `Stochastic`
- `Bollinger Bands`
- `H1` K 线条件
- `ATR` 作为波动过滤

系统二至少满足以下三类条件之一时，尝试开多：

1. 随机指标进入超卖区域，价格位于布林带下轨上方，并处于允许交易时间。
2. H1 上一根 K 线收盘价高于低点加偏移距离，并处于允许交易时间。
3. 价格位于另一组布林带下轨上方，并处于允许交易时间。

系统二支持可选加仓。已有系统二多单时，只有价格相对最近一单回撤达到 `系统二加仓间距`，并且没有超过 `系统二加仓数量`，才允许继续开多。

## 风控与过滤

本重写版实现了以下过滤：

- 最大点差过滤
- ATR 最低波动过滤
- 两套系统合计最大订单数
- 最大账户回撤百分比
- 净值停止百分比
- 亏损后暂停交易
- 连续亏损次数上限
- 周一开始交易小时
- 周五最后开仓小时
- 周五到点全部平仓
- 手动新闻时间过滤

新闻过滤没有接入外部经济日历，因为公开资料无法确认原版使用的新闻数据接口。本版本使用 `手动新闻时间列表` 参数，可以填写：

```text
2026.05.08 12:30;2026.05.08 14:00
```

EA 会在这些时间前后按参数暂停新开仓。

## 主要中文参数

### 主设置

| 参数 | 说明 |
| --- | --- |
| `每跳检测信号` | 为 `true` 时每个 tick 都检查信号；为 `false` 时每分钟检查一次 |
| `最大滑点旧点` | 下单和平仓允许的最大滑点 |
| `自动GMT偏移` | 是否自动读取服务器与 GMT 的偏移 |
| `手动GMT偏移` | 手动设置 GMT 偏移 |
| `使用夏令时` | 是否在手动 GMT 偏移基础上增加夏令时 |
| `允许对冲` | 是否允许两套系统同时存在反向持仓 |
| `挂单距离` | 为 0 时使用市价单；正数使用限价挂单；负数使用止损挂单 |
| `挂单过期分钟` | 挂单超过该分钟数未成交则过期 |
| `订单注释` | 下单时写入订单 comment，可按需要修改 |

## 安装方法

1. 打开 MT4。
2. 点击 `文件 -> 打开数据文件夹`。
3. 进入 `MQL4/Experts`。
4. 复制 `Gold_Chervonets_CN_Rebuild.mq4` 和 `Gold_Chervonets_CN_Rebuild.ex4` 到该目录。
5. 重启 MT4，或者在导航器里刷新 EA。
6. 打开 `XAUUSD` 的 `M15` 图表。
7. 把 EA 拖到图表上，检查参数后运行。

## 回测建议

建议先做三轮测试：

1. 固定手数回测：确认交易频率、最大回撤、平均盈亏和订单行为。
2. 单系统回测：分别只启用系统一、只启用系统二，观察各自表现。
3. 参数优化：重点优化 ATR 阈值、止损止盈、追踪距离、系统二加仓间距和交易时间。

不要直接把默认参数用于实盘。默认参数只是根据公开资料和常见黄金 M15 逻辑设置的可调模板。
