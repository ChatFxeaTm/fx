03 · HY-ZS Range

匯鹰·鹰巢震荡

鹰巢震荡只服务于中枢内部的区间行情。它的使命不是在趋势突破后逆势硬扛，而是在 RANGE 状态下围绕中枢上下沿寻找边界反应。

RANGE 专用边界交易突破停止中低风险

适合行情

价格反复围绕中枢上下沿摆动，区间宽度适中，缺少持续方向，市场状态保持 RANGE。

Logic

震荡交易逻辑

鹰巢震荡的第一原则：不是 RANGE 就不做震荡单。

1. RANGE 状态只有市场状态为 RANGE 时才允许产生候选。
2. 宽度过滤区间过窄容易被点差吞噬，过宽说明风险过大。
3. 边界触发靠近下沿看多，靠近上沿看空。
4. 突破停止出现有效突破后停止新开震荡单。

多单逻辑

  • 中枢有效。
  • 市场状态为 RANGE。
  • 当前价格接近中枢下沿。
  • 没有发生有效下破。
  • 止损放在中枢下沿外侧。
  • 止盈优先看中轨或上沿。

空单逻辑

  • 中枢有效。
  • 市场状态为 RANGE。
  • 当前价格接近中枢上沿。
  • 没有发生有效上破。
  • 止损放在中枢上沿外侧。
  • 止盈优先看中轨或下沿。

停止交易条件

  • 状态转为 BREAKOUT_UP 或 BREAKOUT_DOWN。
  • 状态进入 UP_TREND 或 DOWN_TREND。
  • 区间宽度低于最小值或高于最大值。
  • 点差过大。
  • 同一边界刚刚触发过，冷却未结束。

禁止事项

  • 禁止趋势突破后继续逆势加仓。
  • 禁止无止损高抛低吸。
  • 禁止用马丁补仓或网格扛单。
  • 禁止用震荡策略处理趋势行情。

测试重点

代理测试要重点验证：非 RANGE 状态不交易，突破后停止新开单，区间宽度过滤是否生效，同一边界是否会重复开仓，止损是否始终位于中枢外侧。
