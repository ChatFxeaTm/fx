匯鹰量化 · MT4信号推送教程

如何用 WxPusher 接收 MT4 信号

这份教程给完全不会配置的小白看。按步骤做，MT4 出现信号后，微信里可以收到开仓价、止损、平仓目标、第一止盈位、第二止盈位、第三止盈位。

适用文件
融合指标a_可选交易显示推送EA版.mq4
默认状态
只提醒，不交易
接收方式
WxPusher / 邮件 / 手机推送 / 弹窗

一、先理解这个EA

这个文件是 EA，不是普通指标。EA 可以执行 WebRequest 联网请求，所以适合把 MT4 信号推送到 WxPusher。

默认 AutoTradeOn=false，只显示信号和发送提醒，不会下单。只有你手动改成 AutoTradeOn=true，EA 才会自动交易。

二、准备 WxPusher

标准方式：AppToken + UID

  1. 打开 WxPusher官网，进入管理后台。
  2. 创建应用，复制 AppToken，一般长这样：AT_xxxxxxxxx。
  3. 用微信扫码关注你的应用。
  4. 在后台找到接收人的 UID，一般长这样：UID_xxxxxxxxx。
  5. 把 AppToken 和 UID 填到 EA 参数里。

极简方式：SPT

自己用可以直接拿 SPT_xxx。EA 里只填 WxPusherSPT 也能推送；填了 SPT 后会优先用 SPT。

三、MT4允许WebRequest

  1. MT4 顶部菜单点击 工具 → 选项。
  2. 进入 EA交易 页面。
  3. 勾选 允许 WebRequest 访问列出的 URL。
  4. 新增这条地址：
https://wxpusher.zjiecode.com

只填域名，不要填完整API路径。填错会导致微信收不到。

四、安装EA文件

  1. MT4 点击 文件 → 打开数据文件夹。
  2. 进入 MQL4 → Experts。
  3. 把 融合指标a_可选交易显示推送EA版.mq4 放进去。
  4. 用 MetaEditor 编译。
  5. 回到 MT4，刷新导航器，把 EA 拖到图表上。

五、WxPusher参数怎么填

参数建议说明
WxPusherOntrue打开WxPusher推送。
WxPusherAppTokenAT_xxx标准方式应用Token。
WxPusherUIDUID_xxx接收人UID。
WxPusherSPT可留空如果用SPT模式就填这里。
AlertsOntrueMT4弹窗。
EmailOn按需MT4邮件提醒，终端也要配置邮箱。
PushNotifyOn按需MT4手机推送，终端要绑定MetaQuotes ID。

六、自动交易开关

参数作用建议
AutoTradeOn自动交易总开关默认 false；客户版必须默认关闭。
TradeDirectionMode0多空都做，1只做多，2只做空不确定先用0。
OnePositionOnly同品种同魔术码只保留一单建议 true，防重复开仓。
TradeLots固定手数小资金先0.01。
MaxSpreadPips最大允许点差黄金三位报价35约等于0.35美金。
TradeTPLevel订单止盈目标1=TP1，2=TP2，3=TP3。

打开 AutoTradeOn=true 后，EA 会真实下单。先模拟盘测试点差、滑点、频率、止损止盈，再考虑实盘。

八、常见问题

现象原因处理
微信没收到WebRequest白名单没加检查是否添加 https://wxpusher.zjiecode.com。
提示AppToken为空EA参数没填填写AppToken或SPT。
提示UID为空没有接收人填写WxPusherUID。
不下单AutoTradeOn=false这是安全默认值，想交易才改true。
交易被跳过点差过大/已有持仓/方向限制看MT4底部“专家”日志。

九、推荐配置

只收微信，不交易

WxPusherOn = true
WxPusherAppToken = AT_xxx
WxPusherUID = UID_xxx
AutoTradeOn = false
DrawSignalsOnChart = true
DrawSLTPOnChart = true

微信推送 + 模拟盘自动交易

WxPusherOn = true
WxPusherAppToken = AT_xxx
WxPusherUID = UID_xxx
AutoTradeOn = true
TradeLots = 0.01
OnePositionOnly = true
TradeTPLevel = 3
MaxSpreadPips = 35
