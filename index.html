匯鹰量化Trend 如何使用 WxPusher 接收 MT4 信号

本页面只用于说明 匯鹰量化Trend 指标EA一体版 的 WxPusher 配置。收到的提醒应包含：开仓点、止损点、第一止盈位、第二止盈位、第三止盈位和平仓目标。

一、文件放在哪里

这是指标EA一体版，本质是指标文件，应放在：

MQL4/Indicators/

不要放到 MQL4/Experts/。默认参数 EnableTrading=false 时，只提醒，不自动交易。

二、WxPusher 两种配置方式

方式A：SPT 极简推送

适合自己使用，步骤最少。

  1. 打开 WxPusher 官网：https://wxpusher.zjiecode.com/
  2. 找到极简推送 SPT。
  3. 微信扫码获取自己的 SPT_xxx。
  4. 填入指标参数。
WxPusherOn = true
WxPusherSPT = SPT_xxx
WxPusherAppToken = 留空
WxPusherUID = 留空

方式B：AppToken + UID

适合正式交付客户。

  1. 登录 WxPusher 管理后台。
  2. 创建应用，复制 AppToken，格式类似 AT_xxx。
  3. 用微信扫描应用二维码关注。
  4. 在后台找到接收人的 UID，格式类似 UID_xxx。
  5. 填入指标参数。
WxPusherOn = true
WxPusherAppToken = AT_xxx
WxPusherUID = UID_xxx
WxPusherSPT = 留空

三、MT4 必须允许 WebRequest

  1. MT4 顶部菜单点击 工具。
  2. 点击 选项。
  3. 进入 EA交易 页面。
  4. 勾选 允许 WebRequest 访问列出的 URL。
  5. 添加下面这一条：
https://wxpusher.zjiecode.com

只填这个域名，不要填完整 API 地址，也不要多加空格。

四、推荐参数

参数建议说明
EnableTradingfalse默认只提醒，不自动交易。
WxPusherOntrue开启微信推送。
WxPusherSPTSPT_xxx极简方式填写。
WxPusherAppTokenAT_xxx标准方式填写。
WxPusherUIDUID_xxx标准方式接收人。
MaxSpreadPips35黄金品种建议从 35 开始。
UseTrailingStopfalse移动止损先关闭，测试稳定后再开。

六、常见问题

问题原因处理
微信没收到WebRequest 白名单没加添加 https://wxpusher.zjiecode.com。
提示 Token 或 UID 为空参数没填完整用 SPT，或者填写 AppToken + UID。
日志出现 4060MT4 对指标调用 WebRequest 有限制改用“指标显示 + 桥接EA推送”结构。
挂上后不自动开仓EnableTrading=false这是安全默认值。
