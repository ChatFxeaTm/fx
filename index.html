大盈家EA策略介绍

本策略以 XAUUSD 为主要交易对象，采用“智能网格 + MA方向过滤 + 篮子止盈管理 + 多重账户风控”的组合逻辑。 它不是单纯追涨杀跌，也不是无脑加仓，而是在限制交易频率、点差、保证金和最大持仓数量的前提下，尝试捕捉黄金日内波动中的均值回归与阶段性反弹机会。

一、策略核心定位

匯鹰.量化大盈家EA策略的核心，不是预测每一根K线的方向，而是围绕黄金价格的波动特征，构建一套“分批进场、篮子管理、整体退出”的交易框架。 策略默认只交易 XAUUSD，避免多品种同时运行导致风险叠加。

当前版本更偏向保守型配置：初始手数较低、最大交易数量受限、资金占用比例受限，同时启用了点差过滤、最小保证金保护、最大回撤控制和最旧订单处理机制。 这类设计的目的不是追求短期暴利，而是减少网格策略最容易出现的“越扛越重、越扛越被动”的问题。

二、已经打开的核心功能

1. 智能网格功能 当前启用 UseSmartGrid，策略会结合 RSI 与 RSI均线结构辅助判断入场节奏，不是完全机械式固定距离加仓。
2. MA入场过滤 当前启用 MAEntry，使用长期均线作为市场结构参考，尽量减少在明显不利方向上盲目开仓。
3. 每根K线一次交易 当前启用 EnableOncePerBar，限制同一根K线内重复触发，降低短时间内连续开单带来的滑点和堆仓风险。
4. 利润最大化管理 当前启用 MaximizeProfit，通过移动止盈、利润锁定和分阶段目标管理，让盈利单尽量多走一段。
5. 最旧订单处理 当前启用 UseCloseOldest 与 ForceCloseOldest，当持仓层级上升后，策略会优先处理较早订单，减少老单长期拖累篮子。
6. 账户保护机制 当前启用最小保证金保护，并设置账户亏损停止交易、最大回撤限制、最大点差过滤，控制极端行情下的风险扩张。

四、策略适合的行情

这套策略更适合黄金日内存在来回波动、价格有一定回撤修复能力的行情。震荡行情、缓慢单边中的回调行情、非极端数据时段，通常更适合网格类策略发挥。

它不适合在单边极端行情中裸奔运行，例如重大数据公布、突发地缘事件、流动性断层、跳空高开低开、点差异常放大等环境。 网格策略最怕的不是小亏，而是方向持续错误后仓位不断累积，所以必须把风控参数放在第一位。

五、当前没有作为卖点的功能

当前配置中，CCI、布林带、随机指标、MACD、邮件提醒、假期自动关闭、提前退出、固定止损和断网止损等模块并未作为主要启用功能。 因此，本介绍只围绕已经打开的功能进行说明，不把未启用模块包装成策略优势。

严格讲，这是专业策略介绍必须守住的底线：开了什么就讲什么，没开的功能不要拿来宣传。否则用户预期会错位，后续实盘争议会非常大。

六、风险提示

匯鹰.量化郑重提示：本策略属于自动化交易辅助工具，不构成任何投资建议，也不承诺固定收益。任何历史回测、模拟盘表现、阶段性盈利截图，都不能代表未来实盘结果。

  • 黄金波动剧烈，重大数据和突发消息可能导致滑点、跳空、点差扩大和订单无法按预期成交。
  • 网格策略天然存在持仓累积风险，行情长期单边运行时，浮亏可能快速扩大。
  • 即使设置了最大交易数量、最大回撤、点差过滤和保证金保护，也不能完全避免亏损或爆仓风险。
  • 不同平台的报价、点差、手续费、滑点、杠杆、止损距离限制不同，实盘结果可能与模拟盘明显不一致。
  • 使用前必须先在模拟盘或极小手数真实账户中完成前向测试，确认参数适配自己的平台环境后，再决定是否放大资金。
  • 交易产生的一切盈亏由使用者自行承担，策略开发方不对账户盈利、亏损、爆仓或平台执行差异作任何收益保证。

七、使用建议

建议首次运行时采用低手数、低风险、单品种方式测试，不建议同时挂多个账户、多品种、多策略叠加运行。 对黄金网格EA来说，最重要的不是开单速度，而是账户在极端行情下还能不能活下来。

保守用户应重点关注三项数据：最大浮亏、保证金比例、连续单边行情下的持仓层数。 只看盈利曲线、不看浮亏过程，是评估网格策略时最危险的错误。
